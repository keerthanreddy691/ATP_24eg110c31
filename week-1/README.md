# Day 2 - Dictionaries and Operations

## Topics Covered

### 1. Dictionaries / Objects in JavaScript
Learned how to store data using key-value pairs with JavaScript objects.

### Concepts Learned
- Object creation
- Accessing object values
- Updating object properties
- Adding and deleting properties

### Syntax Practiced

```javascript
const student = {
    name: "Chaitanya",
    age: 20,
    course: "JavaScript"
};
```

---

### 2. Operations on Dictionaries / Objects

#### Accessing Values
Used:
```javascript
student.name
student["age"]
```

#### Updating Values
```javascript
student.age = 21;
```

#### Adding New Properties
```javascript
student.city = "Hyderabad";
```

#### Deleting Properties
```javascript
delete student.course;
```

---

### 3. Object Methods and Iteration

### Practiced:
- `Object.keys()`
- `Object.values()`
- `Object.entries()`

### Iterating Through Objects
```javascript
for (let key in student) {
    console.log(key, student[key]);
}
```
