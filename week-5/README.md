# Day 12 - Flexbox Concept

## Topics Covered

# 1. CSS Flexbox

Learned the Flexbox layout system in CSS for creating responsive and aligned webpage layouts.

---

## Concepts Learned
- `display: flex`
- Flex container
- Flex items
- Row and column layout
- Alignment and spacing

---

## Flexbox Properties Practiced

### Container Properties
- `flex-direction`
- `justify-content`
- `align-items`
- `flex-wrap`
- `gap`

### Item Properties
- `flex-grow`
- `flex-shrink`
- `flex-basis`

---

## Example
```css
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
```

## Concepts Learned
Instead of permanently deleting employee data, implemented soft delete functionality.

- Data is not removed permanently from database

---

## Example
```javascript

```
